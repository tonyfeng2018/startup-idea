## 📋 本次迭代的所有优化点（2026-04-02）

### 1. 七维评估体系（新增2个维度）

**v1-v3**：五维评估（痛点/方案/时机/团队/商业）

**v4 升级**：七维评估，新增：
- **维度六：竞争壁垒** — 必须输出竞品 Feature Matrix + 空白点分析
- **维度七：风险与合规** — Top3风险 + 缓解方案 + 1-5星可缓解性评分

**v5 优化**：合规维度拆解为4个独立子项
- 🔒 隐私合规（个保法/生物特征）
- ⚖️ 偏见公平（算法歧视）
- 📊 防作弊（深度伪造）
- 🤖 AI监管（算法备案/深度合成规定）

---

### 2. 并行搜索强制规则（新增）

**v1-v3**：无强制搜索规则

**v4 新增**：并行搜索6项强制规则
```
搜索1: "[想法] 竞品" → 竞品列表 + 融资金额
搜索2: "[想法] 市场规模 TAM SAM" → 具体数字（亿）
搜索3: "[想法] 渗透率 使用率" → XX%
搜索4: "[想法] 竞品定价" → 具体¥XX/月
搜索5: "[想法] 差评 负面" → Top3痛点 + 占比%
搜索6: "[想法] 融资 2024 2025" → 轮次 + 金额 + 估值
```

**v5 优化**：强制先完成全部6项搜索，再进入第二步，禁止跳步生成报告

---

### 3. 报告诚实度要求（新增）

**v4**：无明确诚实度要求

**v5 新增**：
- 所有缺失数据必须标注 `[待验证]`
- 必须提供具体验证方式（访问XX官网/查阅XX报告/搜索XX关键词）
- 禁止在搜索结果未齐的情况下生成报告
- 报告开头必须声明数据完整性

---

### 4. 搜索全灭时的处理流程（新增）

**v5 新增**：当所有搜索工具全部失效时的标准处理

```
检测到搜索全灭（DuckDuckGo/Bing/Google/web_fetch全部失败）
    ↓
立即停止"假装有数据"
    ↓
基于知识库生成报告
    ↓
所有量化数据标注 [待验证] + 具体验证平台
    ↓
报告开头声明数据完整性
    ↓
末尾提供"根本解决方案"章节
    ↓
提示用户：下次执行前准备好竞品官网截图
```

**已验证的搜索失败场景**：
| 工具 | 错误类型 |
|------|---------|
| DuckDuckGo | Bot detection |
| Google | 重定向到 HK |
| Bing | 重定向 |
| web_fetch | 私有IP封锁 |
| IDC/Gartner | JS挑战拦截 |

**根本解决方案**：
1. 接入火山云搜索API / 百度搜索API / SerpAPI
2. 使用 exec + curl + 代理服务器访问 Google
3. 预先下载关键报告（IDC/艾瑞/淡马锡）存入 memory/

### 7. 报告模块结构（固定模板）

v4-v5 最终版包含以下全部模块（按顺序）：

```
1. 一句话描述
2. ⚡ 核心结论（明确方向）
3. 📊 七维评估表（含评分+量化数据+状态）
4. 🔍 竞品 Feature Matrix（6个竞品）
   - 功能/防作弊/定价/多语言/集成/市场份额/差评痛点
   - 空白点分析（4类）
5. ⚠️ 风险与合规（4项风险+4个子项评分）
6. 📈 SWOT（一句话总结）
7. ✅ 关键假设验证清单（3-5条+验证方式+状态）
8. 💰 竞品融资/退出参考
9. 🏆 差异化方向（2个，各含：
   - 为什么现在能做（最新市场信号）
   - 潜在收入模型（3年预估）
   - 核心优势匹配
   - 关键假设验证）
10. 🏃 最小验证动作（带时间+平台+关键词+具体操作）
11. 📚 数据来源脚注（13条，含验证平台/方式）
12. ⚠️ 重要声明（数据完整性+根本解决方案）
```

---

### 8. 最小动作模板（标准化）

**必须包含字段**：
- # 序号
- 动作 具体描述
- 时间 XX天内
- 平台 具体平台名
- 关键词 具体可搜索的关键词
- 具体操作 可执行的下一步

**格式示例**：
```
| 1 | 约聊银行HR | 7天 | 脉脉/领英 | "银行HR面试官""总行人力资源" | 找10个银行HR深度访谈，问初筛耗时/合规痛点/预算/决策链 |
```

## 🔑 核心经验总结

1. **数据驱动 vs 文档驱动**：每个评分必须有量化依据，无法量化则标注待验证
2. **诚实比完整更重要**：宁可承认数据缺失，也不要假装有数据
3. **搜索失败是常态**：必须设计降级方案，不能依赖单一搜索工具
4. **验证方式比数据更重要**：用户拿着"待验证"标注可以去自行核实
5. **模板是骨架，思考是灵魂**：结构化输出是下限，深度分析才是价值
