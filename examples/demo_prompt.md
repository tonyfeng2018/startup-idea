# startup-idea Demo Prompts

以下是一些可以使用的示例 Prompt：

## 基础验证

```
帮我看看这个想法靠不靠谱：做一个面向HR的AI面试工具
```

```
验证一下这个idea：开发一个帮助中小企业管理供应链的SaaS平台
```

```
帮我分析这个创业想法：做一款面向老年人的健康管理App
```

## 完整分析请求

```
请用7维评估体系帮我分析这个创业想法：
- 想法：做一个面向Z世代的社交电商平台
- 阶段：Idea
- 目标用户：18-25岁大学生和初职年轻人
```

## 快速筛选

```
快速评估这三个想法，选出最有潜力的一个：
1. AI法律顾问
2. 智能招聘平台
3. 企业培训SaaS
```

## 赛道分析

```
帮我看看现在做AI教育硬件这个赛道怎么样？有机会吗？
```

## 输出格式指定

```
帮我验证这个想法，输出格式包括：
1. 7维评分表（带权重和量化依据）
2. 竞品矩阵（至少6个竞品）
3. SWOT一句话总结
4. 关键假设清单（3-5条）
5. 最小验证动作清单（带时间、平台、关键词）
```

---

💡 **提示**: 完整功能需要使用推理模式开启的模型（如 deepseek-v3、glm-4.7），并配合搜索工具获取实时市场数据。
