# idea-validator v5 — 七维评估模板

> **版本**：v5  
> **日期**：2026-04-02  
> **说明**：AI创业想法验证的标准七维评估模板，包含并行搜索规则、报告结构、风险处理机制

---

## 使用方法

1. 复制本模板到新文件
2. 先执行 Step 1（并行搜索6项），等待全部完成
3. 按 Step 2-4 依次填写各模块
4. 所有缺失数据必须标注 `[待验证]`
5. 报告开头声明数据完整性

---

## Step 1：并行搜索（强制执行）

> ⚠️ 必须等全部6项搜索完成后再进入第二步

```
搜索1: "[想法] 竞品"
  → 输出：3-5个竞品 + 融资金额 + 估值

搜索2: "[想法] 市场规模 TAM SAM 2025"
  → 输出：TAM/SAM/SOM具体数字（亿）

搜索3: "[想法] 渗透率 使用率 2025"
  → 输出：XX%渗透率

搜索4: "[想法] 竞品定价"
  → 输出：具体¥XX/月或/年

搜索5: "[想法] 差评 负面"
  → 输出：Top3痛点 + 差评占比%

搜索6: "[想法] 融资 2024 2025"
  → 输出：最新融资轮次 + 金额 + 估值
```

**搜索全灭时**（DuckDuckGo/Google/Bing/web_fetch全部失败）：
- 停止"假装有数据"
- 基于知识库生成报告
- 所有量化数据标注 `[待验证]`
- 报告开头声明数据完整性
- 末尾提供"根本解决方案"章节
